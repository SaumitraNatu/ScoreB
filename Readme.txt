This is a badminton scoreboard app based on REACT with 2 players having their own increment and decrement function in order for score keeping.
The box in the middle resembles a badminton court, where as according to the rules, the court is selected based on the odd-even scoring system in the game.
Some of the states used :
  players : array of players.Each player has a name, score.
  server: server here is referred to the player who serves in each turn. Server can also be viewed as the player who scores the           last point and which player belongs in which side of the court.
scorecard: this holds the scores and the court layout horizontally (as a row).
